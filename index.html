Foresight Sports - Updates

12/5/24

Added a new page:

  • FSX Play Optimizations
    • Graphics Tuning with Lossless Scaling
    • Foresight Provided Help
    • Notes

10/1/24

Added the following changes to FSX Play Scorecards:

  • You can now compare courses in a side by side view
    • Add/Remove courses to Compare with the button under each course
    • The Course Compare panel at the top of the page will show selected courses
    • Share the list of courses with others
    • Click on the title of each course to load the full view down below
    • Drag and drop each course to change the order

9/25/24

Added the following changes to FSX Play Scorecards:

  • You can now identify the par for each hole based on teebox and yardage
    • Par 6 yardages will be underlined
    • Par 5 yardages will be bold
    • Par 4 yardages will be regular
    • Par 3 yardages will be italicized

9/23/24

Added the following changes to FSX Play Scorecards:

  • You can now search for courses in a price range
    • Find courses greater than X and less than Y
  • You can now search for FSX Play courses that are also available in FSX 2020

9/12/24

Added the following changes to FSX Play Scorecards:

  • You can now search for courses by price
    • Find free courses by searching for Price = 0
    • Exclude free courses by searching for Price > 0
    • "Premium" courses are typically those where Price > 150

7/14/24

Added the following changes to FSX Play Scorecards:

  • You can now preview all course hole images at once by clicking "Hole" on the scorecard
  • You can now enter notes for each course
    • Click the "Notes" button under each course to enter formatted text however you like to keep your own personal notes about the course
    • Course notes will persist the next time you come back to the site

7/1/24

Added the following changes to FSX Play Scorecards:

  • Courses now have images for each hole, click on the hole number to see the image
  • Partial course names are now supported via the URL course parameter
    • If more than one course matches (example: ?course=blue), a search will be run to show all matching courses
    • If the partial matches only one course (example: ?course=pebble), that course will be loaded as normal and the URL will be updated to the correct course name

6/24/24

Added the following changes to FSX Play Scorecards:

  • You can now share search criteria for others to see the same results
  • Courses now have background images behind the scorecard, click on the scorecard to see the full size image

6/21/24

Added the following changes to FSX Play Scorecards:

  • The altitude adjustment % for each selected course (if any) will now be shown
  • You can now enter Club Data and if a selected course has an altitude adjustment, adjusted carry values for that course will be shown

6/20/24

Added the following changes to FSX Play Scorecards:

  • Course search now includes: Type
    • Parkland, Links, Desert, Fantasy, etc.
  • You can now mark a course as a favorite and search to find those marked
    • Favorites will persist next time you come back to the site
  • You can now request for a course flyover video!
    • They take a lot of time to create, so please be diligent & patient

6/17/24

Added the following changes to FSX Play Scorecards:

  • Course search now includes: Slope, Rating, and Yardage
    • Courses that have at least one teebox matching these criteria will be returned
  • Update dates added to each course for when data was last extracted

6/14/24

Added the following changes to FSX Play Scorecards:

  • Par search is now a dropdown list of all course par values to choose from
  • Altitude search now supports exact (=), greater than (>), and less than (<) matching to find courses at specific altitudes
  • Flag added to search for courses with links to Flyover videos

6/06/24

Added the following changes to FSX Play Scorecards:

  • Links added for courses to buy them from Foresight Sports
  • Links added for courses that have flyover videos to watch (check back as videos are created)

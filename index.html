Foresight Sports - Updates

6/20/24

Added the following changes to FSX Play Scorecards:

  • You can now mark a course as a favorite and search to find those marked
    • Favorites will persist next time you come back to the site

6/17/24

Added the following changes to FSX Play Scorecards:

  • Course search now includes: Slope, Rating, and Yardage
    • Courses that have at least one teebox matching these criteria will be returned
  • Update dates added to each course for when data was last extracted

6/14/24

Added the following changes to FSX Play Scorecards:

  • Par search is now a dropdown list of all course par values to choose from
  • Altitude search now supports exact (=), greater than (>), and less than (<) matching to find courses at specific altitudes
  • Flag added to search for courses with links to Flyover videos

6/06/24

Added the following changes to FSX Play Scorecards:

  • Links added for courses to buy them from Foresight Sports
  • Links added for courses that have flyover videos to watch (check back as videos are created)

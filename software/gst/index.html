What is the Golf Simulator Toolkit and what does it do?:

The GST was created to provide certain features and provide custom functionality to suppliment golf simulation software.

Here are the current release notes:

What simulation software is supported in the Golf Simulator Toolkit?:

Currently, the following golf simulation software is supported:

  • FSX 2020

The following is being investigated for integration:

  • FSX Play

How do I download the Golf Simulator Toolkit?:

In order to run GST, you will need to install the .NET 9.0 Windows Desktop Framework. The first time you run GST, it will prompt you to download it (~50mb) and easy to install with no choices needed.

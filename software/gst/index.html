What is the Golf Simulator Toolkit and what does it do?:

GST was created for simulation golfers seeking to elevate their game by provides advanced session and shot data, along with video synchronization and editing tools that goes beyond what traditional simulation software offers.

Release notes:

What simulation software is supported in the Golf Simulator Toolkit?:

Currently, the following golf simulation software is supported:

  • FSX 2020

The following is being investigated for integration:

  • FSX Play

How do I download the Golf Simulator Toolkit?:

Latest Version:

System Requirements:

  • Win 10 or newer
  • .NET 9.0 Desktop Framework x64

Notes:

  • The first time you run GST, it will prompt you to download the .NET Desktop Framework if it is not installed (~50mb) and is easy to install with no input needed.
  • Your browser may block the .exe since it is not digitally signed at the moment. Windows Defender might also block it. If so, please reach out.

How do I use the Golf Simulator Toolkit?:

Load a session

Select a shot

Sync swing video(s)
